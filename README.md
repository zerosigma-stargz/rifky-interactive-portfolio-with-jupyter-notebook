# Interactive Portfolio with Jupyter Notebooks

## Overview  
This repository contains an interactive portfolio built with Jupyter Notebooks. It showcases a collection of coding experiments, learning notes, and a professional profile. This portfolio is designed for individuals looking to demonstrate their programming skills and experiences in a visually appealing and accessible format.

## Features  
- **Coding Experiments**: Explore various coding projects and experiments that highlight problem-solving techniques and innovative approaches.
- **Learning Notes**: Documented insights and notes gathered during the learning process, allowing others to benefit from personal experiences.
- **Professional Profile**: An overview of qualifications, skills, and experiences, serving as a digital resume.

## Getting Started  
To get started with building your interactive portfolio, clone the repository and start exploring the notebooks. Ensure you have Jupyter installed to run the notebooks locally.

## Requirements  
- Python 3.x  
- Jupyter Notebook  
- Required libraries (listed in requirements.txt)
